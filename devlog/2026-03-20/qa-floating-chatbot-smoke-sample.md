## 8. 샘플 시나리오 결과 예시
### 시나리오 A: 참고 문헌 기반 질의
- 질문: 최근 플라즈마 진단 논문 기준으로 핵심 이슈 정리해줘
- 예시 정상 결과:
  - [x] 사고과정 단계가 진행된다.
  - [x] 참고 문헌 목록이 생성된다.
  - [x] 유사도 요약이 표시된다.
  - [x] 최종 답변이 별도 카드로 정리된다.
- 기대 화면 메모: 참고 문헌 카드 2~5개, 유사도 요약 1개, 최종 답변 카드 1개 구성

### 시나리오 B: 페이지 이동형 질의
- 질문: 실험 데이터셋 페이지로 이동해서 조건-결과 관점으로 정리해줘
- 예시 정상 결과:
  - [x] tool 카드에 이동 관련 로그가 보인다.
  - [x] 페이지 이동 또는 handoff 안내가 자연스럽게 표시된다.
  - [x] 최종 답변이 분리되어 유지된다.
- 기대 화면 메모: tool 로그에 navigation 관련 기록이 보이고, 사용자 입장에서 이동 사유를 이해할 수 있어야 한다.

### 시나리오 C: 보고서형 질의
- 질문: 이 주제를 보고서 초안 형태로 정리해줘
- 예시 정상 결과:
  - [x] 단계 설명이 보고서 작성 흐름에 맞게 갱신된다.
  - [x] 최종 답변이 구조화된 텍스트로 표시된다.
  - [x] UI 깨짐 없이 긴 응답을 수용한다.
- 기대 화면 메모: 제목/개요/핵심 포인트/후속 제안처럼 구조화된 답변이 자연스럽게 읽히면 정상으로 본다.
